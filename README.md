# Api-Table-FIPE 🚗💰

A simple and efficient application to query vehicle price data from the **FIPE Table** (Foundation Institute for Economic Research) in Brazil.

---

### 📝 Description
This project provides a programmatic way to access average market prices for cars, motorcycles, and trucks. It is designed for developers who want to integrate automotive market data into their own systems or for those conducting data analysis on vehicle depreciation and pricing in the Brazilian market.

### ✨ Key Features
* 🔍 **Brand Listing:** Retrieve all vehicle brands registered in the database.
* 🚘 **Model Search:** Filter and find specific models for any selected brand.
* 📅 **Year Filtering:** Access historical or current pricing based on the vehicle's model year.
* 💵 **Real-time Data:** Get the most recent market values directly via API integration.

### 🚀 Getting Started

#### Prerequisites
Before running the project, ensure you have the environment set up (e.g., **Python 3.x** or **Node.js**).

#### Installation
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Thy281/Api-Tabela-FIPE.git](https://github.com/Thy281/Api-Tabela-FIPE.git)
    ```
2.  **Navigate to the directory:**
    ```bash
    cd Api-Tabela-FIPE
    ```
3.  **Install dependencies:**
    ```bash
    # For Python projects:
    pip install -r requirements.txt

    # For Node.js projects:
    npm install
    ```

#### Usage
Run the main script to start querying:
```bash
python main.py  # or npm start
